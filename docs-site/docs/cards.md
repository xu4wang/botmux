# 实时流式卡片

每轮对话生成一张实时更新的飞书卡片，是你在手机/飞书上**感知并操控 CLI** 的主窗口。

![实时流式卡片](https://magic-builder.tos-cn-beijing.volces.com/uploads/1780419090587_img_v3_0212a_553ca347-4a93-491f-a2ef-30d00a374cdg.jpg)

- **终端画面实时截图刷新到卡片**：xterm 无头渲染成图，**原样还原 CLI 的 TUI**（边框、配色、光标都在），不再把输出转成 Markdown。可一键「显示 / 隐藏输出」「导出文字」「上 / 下翻屏」。
- **状态实时指示**：🟡 启动中 → 正在分析 → 🔵 工作中 / 执行中 → 🟢 等待输入；额度用满会标「限额已达 · 可重试」。
- **卡片上直接操作**：打开（可写）终端、🔑 取操作链接、重启 / 关闭 / 接管会话、重发上一条任务。
- **每轮一张新卡片**：上一轮卡片冻结存档，对话历史清晰可回溯；会话用 [`/relay`](/relay) 搬到别的群后，原卡片也会自动冻结为存档。
- **关闭时给「可恢复」卡片**：附上 CLI 原生 resume 命令，随时点回来继续。

> 卡片正文是终端画面的**实时截图（图片）**，不是文本渲染。CLI 主动发的消息（通过 `botmux send`）则是独立的富文本 / 图文消息，可带图片、文件、@mention。
